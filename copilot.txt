1. Use Open-Meteo Geocoding API (no key needed):

Endpoint:
https://geocoding-api.open-meteo.com/v1/search?name={city}&count=1
Example:
https://geocoding-api.open-meteo.com/v1/search?name=Berlin&count=1
Returns latitude and longitude for the city.
2. Use the coordinates with Open-Meteo Weather API:

Example:
https://api.open-meteo.com/v1/forecast?latitude={lat}&longitude={lon}&current_weather=true
3. Implementation steps:

Add a search bar (input for city/country) in your HTML.
In JavaScript, fetch coordinates using the geocoding API when the user submits.
Use the returned coordinates to fetch weather data as usual.
Sample JavaScript function for geocoding:
async function getCoordinates(city) {
    const url = `https://geocoding-api.open-meteo.com/v1/search?name=${encodeURIComponent(city)}&count=1`;
    const response = await fetch(url);
    const data = await response.json();
    if (!data.results || data.results.length === 0) throw new Error("Location not found");
    return { latitude: data.results[0].latitude, longitude: data.results[0].longitude };
}

You can then use these coordinates in your weather API call.
